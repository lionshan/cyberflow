## 一键评论需求实现

1. 开发 vue 组件页面 OneClickComment 10H
   AI 智能体选择
   一键回复按钮，关闭 运行 结束状态
   回复评论数据列表展示当前发布的回复列表

2. popup 页面使用获取当前 tabId 5H
   识别是否可以进行一键回复

```js
//获取当前tabId和url
chrome.tabs.query({ active: true, currentWindow: true }, (tabs) => {
    const tabId = tabs[0].id;
    // 发送消息给 content script
    chrome.tabs.sendMessage(tabId, { action: "doSomething", data: { foo: 123 } });
});
```

发送一键评论消息到 content.ts 执行

3. content.ts 实现方法 OneClickComment 10H
   回复当前推文帖子
   然后间隔随机时间
   回复评论内容中的其他帖子，需要记录回复过的帖子 id(最大回复 10 条或者当前列表都回复完成)
   然后间隔随机时间
